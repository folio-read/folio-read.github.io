# Folio

Folio is a quiet place to keep the things you mean to read.

You know how it goes: you find an article, you know you're not reading it
right now, so you leave it open in a tab and tell yourself you'll come back
to it. You won't. Folio is a small fix for that one specific habit — paste
a link, it pulls the title, a short excerpt, the cover image, and a rough
read time, and drops it into a list you can actually come back to later.

## What it's for

- **An inbox that's actually a queue.** Everything you save starts in your
  Inbox. Once you start reading something, mark it as Reading. Once you're
  done, archive it. Three states, no folders to invent.
- **Tags you make up yourself.** No fixed categories — whatever labels make
  sense to you, in whatever colors you like.
- **Highlights that stay with the article.** Pull out a line worth
  remembering, add a note next to it if you want, and it stays attached to
  that piece for good — not scattered across some separate notes app.
- **Search that searches everything.** Titles, authors, excerpts, and your
  own highlights, from one search box, not four filters you have to
  remember to set.
- **A small stats page**, mostly so you can see, in a fairly unflattering
  way, how much you save compared to how much you actually read.

## What it isn't

This isn't Pocket or Readwise with a subscription and a team behind it —
it's a personal tool, built for one person's reading habits. 

## Using it

Paste a link on the Add screen, glance at the preview it builds, save it if
it looks right. 

That's it. It's meant to stay small.
